Was ist Node.js?

  • JavaScript-basiertes I/O Framework, das auf Googles V8-Engine aufbaut
  • Ereignisgesteuerter Programmablauf, der nicht durch
    Ein-/Ausgabeoperationen (z.B. Datenbankzugriffe) blockiert wird
  • alle Funktionen mit I/O-Operationen besitzen eine Rückruffunktion (Callback)
  • Schlanker Kern, Basismodule und über 13000 weitere Pakete, die über einen Paketmanager nachgeladen werden können

Hallo Welt

Ausgabe auf der Konsole:

console.log("Hallo Welt");

Webserver:

var http = require("http");

http.createServer(function(request, response) {
  response.writeHead(200, {"Content-Type": "text/plain"});
  response.write("Hallo Welt");
  response.end();
}).listen(3000);

Module für WebApps

  • Application Framework: Express
  • Template Engine: Jade
  • CSS Pre-Processor: Stylus
  • WebSocket: Socket.IO
  • File Monitor: nodemon

          $ express -c stylus
